System Requirements:
Operating System:
Win32 v4.0 or higher (Windows 95-98, Windows Millennium, Windows NT 4.0, Windows 2000, XP, and 2003)

Merge Support:
In order for iCode+Generator to support File Merge you will need to install.
WinMerge. You can download a copy of this excellent open source application at http://winmerge.sourceforge.net/.
